⚙️ Jenkins Installation & Ngrok Guide

🖥️ Step 1: Install Java (Required)

Jenkins requires Java. Run the following commands:

sudo apt update
sudo apt install openjdk-11-jdk -y
java -version

📥 Step 2: Install Jenkins

Run these commands in your terminal (for Ubuntu/Debian):

wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -
sudo sh -c 'echo deb https://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'
sudo apt update
sudo apt install jenkins -y

🚀 Step 3: Start Jenkins Service

sudo systemctl start jenkins
sudo systemctl enable jenkins
sudo systemctl status jenkins

🔓 Step 4: Unlock Jenkins

Visit http://localhost:8080 in your browser.

To get the admin password, run:

sudo cat /var/lib/jenkins/secrets/initialAdminPassword

🛠 Step 5: Install Recommended Plugins

  • Follow on-screen instructions
  • Install suggested plugins
  • Create your admin user
✅ Tip: If port 8080 is in use, change the port in /etc/default/jenkins

🌐 Optional: Expose Jenkins Using Ngrok

1. Download Ngrok and install it.
2. Run this command: 
ngrok http 8080
3. Use the generated https://*.ngrok.io URL to access Jenkins from anywhere.

✅ Final Check

  • Jenkins running at http://localhost:8080
  • Ngrok exposes it: https://xyz.ngrok.io
  • Ready to create jobs, setup CI/CD, and integrate GitHub

💡 Best Advice

  • Keep Jenkins updated for security patches
  • Use pipelines over freestyle for advanced workflows
  • Store Jenkins jobs in a versioned folder with backup
🔐 Secure Your Jenkins: Install role-based access control plugin and always use HTTPS (behind Nginx/Apache).


🎉 You're now ready to build CI/CD pipelines with Jenkins!
